저는 특성화고등학교에서 **관광학**을 전공하고 대학교에서 **영문학**을 전공하면서, 저는 언어를 도구 삼아 낯선 사람들의 긴장을 깨고 마음을 열게 하는 일에 뿌듯함을 느껴왔습니다.
영문학을 전공하며『햄릿』같은 근대 비극부터 『세일즈맨의 죽음』같은 현대 희곡까지 다양한 작품을 읽어왔습니다. 특히 셰익스피어의『햄릿』을 읽으며 햄릿 특유의 반어법과 은유법 이면에 담긴 숨은 함의를 읽어내며 흥미로움을 느꼈습니다. 이를 실제 현장에서도 적용하기 위해 고등학교 1학년 때부터 요식업과 서비스 분야에서 꾸준히 아르바이트를 하며 텍스트 이면을 분석하듯 하객들의 미묘한 표정과 비언어적 신호를 살폈고, 특히 어르신들과 혼주 내외분께는 요청하시기 전 먼저 다가가 필요한 것을 챙겨드리는 능동적인 서비스를 실천해왔습니다. 이러한 경험을 통해 **언어 속 함의를 읽어내는 것은 AI가 계산할 수 없는 인간만의 영역**임을 깨닫게 되었습니다.
